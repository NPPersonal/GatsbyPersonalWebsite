## Introduction

---

**Style transfer app allow you to transform a real photo into a vangogh
panting style photo.**

<br />

## Motivation & Purpose

---

**To learn vangogh's panting style and then create an image from reality with the style
is quite difficult to achieve and take a long time.**

**A robot that learn vangogh's panting style then transform real photo into vangogh
style image. This speed up the whole process.**

<br />

## Detail

---

The model was trained with
dataset from [Kaggle](https://www.kaggle.com/datasets/suyashdamle/cyclegan?select=vangogh2photo).

The model use [CycleGAN](https://www.tensorflow.org/tutorials/generative/cyclegan) to achieve
style transfer for real photo.

Frameworks were used to build this app:

- **[Tensorflow](https://www.tensorflow.org/)** : deep learning framework for the model
- **[Streamlit](https://streamlit.io/)** : frontend app
