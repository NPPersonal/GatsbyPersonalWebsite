## Introduction

---

A website let you explore movies. The website give you information about movies.
You are able to see what is popular movies currently, top rated movies, Trending and
what movies are screening now.

<br />
<br />

Furthermore the webiste also tell you what lanaguages are available for a movie, how
long is the movie, who are the casts and trailers.
And of course the reviews for the movie from many peoples.

<br />
<br />

## Motivation & Purpose

---

The purpose to make this website is to learn how to use [Next.js](https://nextjs.org/)
a popular framework.

<br />
<br />

Through the learning journey, the website was produced.

<br />
<br />

## More

---

The website use Next.js as frontend. Instead of using backend server framework
such as Express.js, it leverage the feature serverless from Next.js.

<br />
<br />

serverless also known as serverless functions. Functions that only wake up and
work when requests come in and put to sleep when no longer needed. Thus there is
no need to run a server continuously.

<br />
<br />

Frameworks were used to build this website:

- [Next.js](https://nextjs.org/) : **Next.js framework**
- [Material UI](https://mui.com/material-ui/) : **frontend UI framework**
- [Themoviedb](https://developer.themoviedb.org/docs/getting-started) : **movies data source**
- [Cypress](https://www.cypress.io/) : **Testing framework**
